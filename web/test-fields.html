🧪 Prueba de Clasificación de Campos

📖 Cómo Probar el Sistema

  1. Carga esta página en Chrome con la extensión instalada
  2. Abre la consola del navegador (F12 → Tab "Console")
  3. Ejecuta el comando: FFYDebug.printReport()
  4. Verás qué campos detecta la extensión y con qué confianza
  5. Prueba con diferentes formatos para ver cómo el sistema maneja variaciones
